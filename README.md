🎬 Netflix Analytics Dashboard (Excel Project)

This project focuses on building a complete Netflix Analytics Dashboard from scratch using Excel, designed to simulate a real-world data analytics workflow.

👉 Watch the full step-by-step implementation on YouTube: [https://youtu.be/yVjOOR_lYJc]

🚀 Project Objective

The goal of this project is to move beyond theory and build a practical, industry-level dashboard that demonstrates how data can be transformed into meaningful insights.

📊 Key Features
Data cleaning and preprocessing in Excel
Creation of KPIs for business insights
Use of Pivot Tables for data summarization
Interactive dashboards using charts and slicers
Clear data storytelling for decision-making
